Dokumentasi API Node VPN SRPCOM V5

API SRPCOM V5 adalah gerbang remote API berbasis **Flask Python** (`xray-api.py`) yang berjalan di balik reverse proxy **Caddy Web Server**. Layanan ini dirancang khusus untuk mempermudah developer dalam membangun bot manajemen Telegram otomatis atau mengintegrasikan VPS Multiport V5 ke dalam website panel penjualan kustom Anda.

Base URL Server: yourvps.com
Protokol Keamanan: HTTPS (SSL Caddy Otomatis)
**Keamanan Terpusat**: Seluruh endpoint API beroperasi secara non-interaktif dan aman. Pastikan API diaktifkan melalui panel server VPS di Terminal Anda sebelum memanggil request remote.

Autentikasi Keamanan

Mengamankan remote request dari pihak luar menggunakan token header.

Semua panggilan endpoint API dilindungi menggunakan sistem token rahasia yang dikirimkan melalui header HTTP. API Key ini dapat diatur ulang pada file konfigurasi VPS Anda di `/usr/local/etc/xray/api_key.conf`.

HTTP Header Wajib

Nama Header x-api-key
Nilai / Value SANGATRAHASIA123 (Sesuai isi config VPS)
Configuration Engine

Dynamic API Sandbox HUD

Masukkan Domain VPS dan API Key Anda di bawah ini. Seluruh request URL, perintah cURL, dan contoh kode integrasi di halaman ini akan terupdate secara otomatis dan siap untuk langsung disalin!

Xray Core API (VMESS, VLESS, TROJAN)

Mengelola akun Xray core berbasis Websocket (Port 80/443).

POST /srpcom/add-<protocol>ws
Protocol: vmess, vless, trojan

Membuat akun user baru dengan limitasi IP (Multi-login) dan kuota (GB).

curl -X POST https://yourvps.com/srpcom/add-vmessws \
  -H 'Content-Type: application/json' \
  -H 'x-api-key: SANGATRAHASIA123' \
  -d '{
    "user": "developer_test",
    "exp": 30,
    "limit_ip": 2,
    "limit_quota": 50
  }'
{
  "stdout": "Pembuatan akun baru berhasil...\nRemarks : developer_test\n...",
  "stdout_tg": "Pembuatan akun baru berhasil...\nRemarks : `developer_test`\n..."
}

Integrasi Bot Telegram Kustom Anda

Panduan praktis membangun bot Telegram kustom menggunakan library Python Telebot (`pyTelegramBotAPI`).

Anda dapat memanfaatkan library Python untuk membuat bot Telegram interaktif agar reseller atau pembeli Anda dapat melakukan transaksi mandiri (transaksional bot) secara otomatis. Di bawah ini adalah contoh kode bot Python siap pakai: 

import telebot
import requests

# Masukkan Token Bot Telegram Anda dari @BotFather
BOT_TOKEN = "YOUR_TELEGRAM_BOT_TOKEN"
bot = telebot.TeleBot(BOT_TOKEN)

# Parameter Konfigurasi API VPS SRPCOM V5 (Ter-sync dinamis dari HUD)
API_URL = "https://yourvps.com"
API_KEY = "SANGATRAHASIA123"

HEADERS = {
    "Content-Type": "application/json",
    "x-api-key": API_KEY
}

@bot.message_handler(commands=['start', 'help'])
def send_welcome(message):
    welcome_text = (
        "🤖 *Selamat Datang di Bot VPS Manager!*\n\n"
        "Gunakan perintah di bawah ini:\n"
        "• `/addvmess ` - Buat akun VMess\n"
        "• `/addssh  ` - Buat akun SSH & OVPN\n"
        "• `/detailvmess ` - Cek konfigurasi VMess\n"
        "• `/listvmess` - Daftar seluruh akun VMess aktif\n"
        "• `/vpsstatus` - Pantau performa Server (CPU/RAM)"
    )
    bot.reply_to(message, welcome_text, parse_mode="Markdown")

@bot.message_handler(commands=['addvmess'])
def handle_add_vmess(message):
    args = message.text.split()
    if len(args) < 2:
        bot.reply_to(message, "❌ *Gagal*: Gunakan format `/addvmess [username]`")
        return
        
    username = args[1]
    bot.reply_to(message, f"⌛ Sedang memproses pembuatan akun VMess: `{username}`...")
    
    # Panggil Endpoint API SRPCOM
    response = requests.post(
        f"{API_URL}/srpcom/add-vmessws",
        json={"user": username, "exp": 30, "limit_ip": 2, "limit_quota": 50},
        headers=HEADERS
    )
    
    if response.status_code == 200:
        bot.reply_to(message, response.json().get("stdout_tg"), parse_mode="Markdown")
    else:
        bot.reply_to(message, f"❌ *Error*: Gagal menghubungi API VPS (Status: {response.status_code})")

@bot.message_handler(commands=['listvmess'])
def handle_list_vmess(message):
    bot.reply_to(message, "⌛ Mengambil data akun dari VPS...")
    response = requests.get(f"{API_URL}/srpcom/list-accounts/vmess", headers=HEADERS)
    
    if response.status_code == 200:
        bot.reply_to(message, response.json().get("stdout_tg"), parse_mode="Markdown")
    else:
        bot.reply_to(message, "❌ *Gagal*: Tidak dapat mengambil daftar akun aktif.")

@bot.message_handler(commands=['vpsstatus'])
def handle_status(message):
    response = requests.get(f"{API_URL}/srpcom/sys-resource", headers=HEADERS)
    if response.status_code == 200:
        bot.reply_to(message, response.json().get("stdout"), parse_mode="Markdown")
    else:
        bot.reply_to(message, "❌ *Gagal* mengambil status server.")

print("🤖 Bot Telegram Kustom sedang berjalan...")
bot.infinity_polling()
